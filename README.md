# 酒店管理系统简介 java909

## 项目概述

酒店管理系统是基于springboot和vue的前后端分离架构，提供了一系列全面实用的功能模块，旨在提升酒店管理效率和服务质量。

## 技术栈

- 后端：Spring Boot + MyBatis + MySQL
- 前端：Vue + Element + MuseUI

## 功能模块

### 移动端

- 查看酒店房型
- 预订房间
- 支付订单
- 我的订单
- 订单详情

### 后台管理系统

- 房间类型管理
- 房间信息管理
- 预订方式管理
- 操作员管理
- 服务员管理
- 入住登记
- 退房登记

## 系统角色

系统包含以下角色：

- 客户：使用移动端进行酒店房间的查看、预订、支付及订单管理。
- 管理员：通过后台管理系统进行房间信息、预订方式、用户及订单的管理。

## 运行环境

- 后端：Java 1.8+，MySQL 5.7+
- 前端：现代浏览器支持
